# Cafe Appointment Booking App – UI/UX Design

This project is a mobile UI/UX design for a cafe appointment booking application created using Figma.

##  Project Goal
To design a clean, modern, and user-friendly mobile interface that allows customers to easily book tables, select hosts, and schedule appointments.

##  Key Features
- Simple login flow
- Event selection screen
- Staff selection interface
- Date & time booking
- Booking confirmation UI

##  Tools Used
- Figma
- Auto Layout
- Components
- Prototype interactions
